## Choosing the Right Model

- **Highest quality, no time constraints?** Go with `flux-2-max` or `gemini-imagen-4`
- **Best speed/quality balance?** Try `flux-2-pro` or `kling-text-to-image`
- **Need text in the image?** Use `flux-kontext-pro` (best at rendering text)
- **High volume / rapid iteration?** Use `nano-banana-pro`
- **Creative / artistic styles?** Use `seedream-v4-5`
