# Introduction to Platform APIs

> Platform APIs provide programmatic access to platform resources including model metadata, usage tracking, analytics, and fal Assets.

- **[Platform APIs for Models](/api-reference/platform-apis/for-models)** — Access model metadata, pricing information, usage tracking, and analytics for Model APIs

- **[Platform APIs for Assets](/api-reference/platform-apis/for-assets)** — Browse, search, upload, and organize generated or uploaded media in fal Assets

- **[Platform APIs for Storage](/api-reference/platform-apis/for-storage)** — Control access, sign URLs, and manage lifecycle settings for fal CDN files

- **[Platform APIs for Serverless](/api-reference/platform-apis/for-serverless)** — APIs to manage and monitor your serverless applications

- **[Platform APIs for Compute](/api-reference/platform-apis/for-compute)** — Manage compute instances with lifecycle control and monitoring capabilities

- **[Platform APIs for Workflows](/api-reference/platform-apis/for-workflows)** — List and retrieve workflow metadata and definitions

- **[Platform APIs for Accounts](/api-reference/platform-apis/for-accounts)** — Access account billing information and FOCUS-compliant cost reports

> **📝  Note:** For executing models and generating content, see [Model APIs](/documentation/model-apis/overview).
