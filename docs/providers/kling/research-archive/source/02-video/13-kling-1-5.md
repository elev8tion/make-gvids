# Kling 1.5

**Source:** https://kling.ai/document-api/api/video/1-5

Uses same endpoints with `model_name: "kling-v1-5"`:

- **POST** `/v1/videos/text2video` — Text to Video
- **POST** `/v1/videos/image2video` — Image to Video
- **GET** endpoints for querying

Same parameter structure. See Kling 2.6 for full parameter details.
