# Kling Image 1.5

**Source:** https://kling.ai/document-api/api/image/1-5/image-generation

Uses same endpoint structure with `model_name: "kling-v1-5"`:

- **POST** `/v1/images/generations`

Model v1-5 supports `image_reference` (subject/face), `image_fidelity`, and `human_fidelity` parameters.

See [Kling Image 3.0 & 3.0 Omni](./image/01-kling-image-3-0-omni.md) for parameter details.
