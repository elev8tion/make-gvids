# Callback Protocol

**Source:** https://kling.ai/document-api/api/get-started/callbacks

## Overview

For async tasks (image generation / video generation / virtual try-on), if you actively set the `callback_url` when creating a task, the server will actively notify you when the task status changes.

## New Callback Function

**Applicable to:** Kling Video 3.0 Turbo and subsequent new models.


### Fields

| Field | Description |
|---|---|
| id | The task ID being queried |
| status | Task status: submitted, processing, succeeded, failed |
| message | Status info, shows failure reason when task fails |
| create_time | Task creation time, Unix timestamp (ms) |
| update_time | Task update time, Unix timestamp (ms) |
| external_id | Custom task ID (if any) |
| outputs | Array of generated results (type: image, video, audio, element, voice) |
| billing | Task consumption information |

### Output Types

- **video**: Video ID, URL, watermark URL, duration
- **image**: URL, watermark URL, group_id (for group images)
- **audio**: MP3/WAV URLs, durations
- **voice**: Voice ID, name, preview URL, source
- **element**: Element ID, name, description, type, materials, tags

> **Note:** Generated results are cleared after 30 days. Save them promptly.

## Legacy Callback Function

**Applicable to:** Kling 3.0 Omni and earlier models.

```json
{
  "task_id": "string",
  "task_status": "string",
  "task_status_msg": "string",
  "created_at": 1722769557708,
  "updated_at": 1722769557708,
  "final_unit_deduction": "string",
  "task_info": {
    "parent_video": {
      "id": "string",
      "url": "string",
      "duration": "string"
    },
    "external_task_id": "string"
  },
  "task_result": {
    "images": [
      {
        "index": 0,
        "url": "string"
      }
    ],
    "videos": [
      {
        "id": "string",
        "url": "string",
        "duration": "string"
      }
    ]
  }
}
```

### Fields

| Field | Description |
|---|---|
| task_id | Task ID generated by system |
| task_status | submitted, processing, succeed, failed |
| task_status_msg | Failure reason (e.g., content risk control) |
| created_at | Task creation time, Unix ms |
| updated_at | Task update time, Unix ms |
| final_unit_deduction | Deduction units of task |
| task_info | Task creation parameters |
| task_result | Result with images[] and videos[] arrays |
