# Video Effects

**Source:** https://kling.ai/document-api/api/effects/video-effects

## Overview

Video Effects API for applying post-processing effects to generated or uploaded videos.

## Endpoints

- **POST** `/v1/videos/effects` — Create video effect task
- **GET** `/v1/videos/effects/{id}` — Query effect task
- **GET** `/v1/videos/effects` — List effect tasks

Standard async task pattern with callback support.
