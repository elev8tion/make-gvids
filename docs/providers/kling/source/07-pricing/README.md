## Key Facts

- **Unit price**: 1 Unit = **$0.14 USD** (standard rate)
- **Billing types**: Per second (video), Per image (image), Per request/call (misc)
- **Video resolutions**: 720P, 1080P, 4K
- **Image resolutions**: 1K, 2K, 4K
- **3 pricing sections**: Video (15+ models), Image (11+ models), Effects (~200 templates)
