# Kling 2.5 Turbo

**Source:** https://kling.ai/document-api/api/video/2-5-turbo

Uses same endpoints as Kling 2.6 with `model_name: "kling-v2-5-turbo"`:

- **POST** `/v1/videos/text2video` — Text to Video
- **POST** `/v1/videos/image2video` — Image to Video
- **GET** `/v1/videos/text2video/{id}` | `/v1/videos/text2video` — Query
- **GET** `/v1/videos/image2video/{id}` | `/v1/videos/image2video` — Query

Same parameter structure as Kling 2.6. See that document for full parameter details.
